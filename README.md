📦 Novel Tool
A cross-platform lightweight tool designed by a Vietnamese developer to help users read, create, edit, and convert EPUB/TXT novels easily, even without technical knowledge.

✨ Features
•Read and edit EPUB/TXT novels easily
•Custom regex-based rendering (MDT – Máy Đọc Truyện engine)
→ see MDT.html for details
•Convert between multiple versions
•Lightweight and fully offline-capable
•Cross-platform support

💻 Supported Platforms
•Windows
•Linux / macOS
•Android (via Chrome)
•Optional localhost mode (recommended)

📁 File Types
•.bat, .ps1 → Windows tools
•.sh → Linux / macOS tools

⚙️ Versions
📌 Version 1 — MDT Core Engine
•Integrated inside MDT.html
•Single-pipeline processing
•Regex-based rendering system
•Full read & edit workflow
•Tight coupling between logic and UI

📌 Version 2 — Extended Engine
•Fully separated architecture
•Custom chapter detection system
•Splitter-based chapter separation
•Border-based UI formatting:
	•Title
	•Description
	•Metadata
•Improved readability for long novels
•Designed for extensibility

🧠 Notes
•Rendering uses custom regex rules (MDT engine)
•More rules will be added in future updates
•Workflow: Load → Convert → Edit → Export
•CJK text (Chinese/Korean/Japanese) may not render correctly
•Project is still early-stage

🌐 Offline Support
•Fully works offline
•Localhost mode recommended for stability

🤝 Contribution
•Suggestions and improvements are welcome.

📱 Android Setup
Requirements
•Chrome or any modern browser
•Local server app (SHTTPS / KSWEB / similar)
•Termux (optional)
•Bluetooth keyboard (optional)

Steps:
•Install SHTTPS
•Extract tool folder
•Set root directory to project folder
•Enable localhost mode
•Open index.html
•Start using tool
